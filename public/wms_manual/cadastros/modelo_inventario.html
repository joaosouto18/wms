Cadastros -> Modelo de Inventário

O modelo de inventário é usado como base para o novo processo de inventário. Aqui define-se quais serão os parâmetros que o inventário deverá seguir. É possível definir N modelos de inventários com diversas configurações possíveis.

Clique no menu CADASTROS, em seguida clique em MODELO DE INVENTÁRIO.

Para cadastrar um modelo de inventário, clique em NOVO MODELO DE INVENTÁRIO, conforme item 1 da figura abaixo.
Para EDITAR/EXCLUIR um modelo de inventário, posicione o mouse sobre a seta no início da linha, em seguida clique em EDITAR ou EXCLUIR, conforme item 2 da figura abaixo.
.

Na tela de cadastro, você definirá as características desse modelo de inventário.

  • Número de contagens: número necessário de contagens a serem realizadas.
  • Modelo de inventário ativo: informa se esse modelo de inventário está ativo ou inativo.
  • Controla validade:
    • Não controla: não controla a data de validade durante o inventário.
    • Apenas confere: define a data digitada no inventário como a correta.
    • Confere e valida: confere a data informada no inventário com o valor do sistema.
  • Compara estoque: define se a contagem levará em conta o estoque atual.
  • Um usuário pode fazer N contagens: define se o mesmo usuário poderá efetuar mais de uma contagem no inventário.
  • Contar tudo no endereço: define se todos os itens que estão no endereço deverão ser contados.
  • Contar volumes separadamente: define se os volumes de um produto serão contados separadamente.

Em seguida clique em SALVAR
